Генератор случайных паролей

Автор: Фёдорова Елизавета Антоновна

Описание
Генератор случайных паролей с графическим интерфейсом на Python (Tkinter). Позволяет настраивать длину и состав пароля (буквы, цифры, спецсимволы), сохранять историю сгенерированных паролей в JSON и копировать пароль в буфер обмена.

Обеспечивает надёжную работу: проверяет диапазон длины (8–32), создаёт файл истории при первом запуске и корректно обрабатывает ошибки чтения JSON.

Требования
Python 3.6+.
Библиотеки:
pyperclip (для работы с буфером обмена).
Tkinter входит в стандартную библиотеку Python.
Установка зависимостей
Установите необходимые пакеты из файла requirements.txt:
bash
Копировать
pip install -r requirements.txt
(Опционально) Убедитесь, что у вас установлен Python и он добавлен в PATH.
Запуск приложения
Откройте терминал в папке проекта.
Запустите скрипт:
bash
Копировать
python main.py
Если у вас несколько версий Python, используйте python3 main.py.

Возможности
Настройка длины пароля: от 8 до 32 символов.
Выбор типов символов: буквы (A–Z, a–z), цифры (0–9), спецсимволы (!@#$%...).
Генерация криптографически стойких паролей с помощью модуля secrets.
Сохранение истории: последние 20 паролей хранятся в файле history.json.
Копирование в буфер обмена: одним нажатием кнопки.
Надёжность: приложение не «падает» при повреждении файла истории или первом запуске.
