Random Password Generator

Автор: Фёдорова Елизавета Антоновна

Описание

Генератор случайных паролей с графическим интерфейсом на Python (Tkinter). Позволяет настраивать длину и состав пароля (буквы, цифры, спецсимволы), сохранять историю сгенерированных паролей в JSON и копировать пароль в буфер обмена.

Требования

- Python 3.6+
- Библиотеки: tkinter, pyperclip (устанавливается через pip: `pip install pyperclip`)

Запуск

1. Клонируйте репозиторий:
  
   git clone <ссылка_на_репозиторий>
   
2. Перейдите в папку проекта:
   
   cd password_generator
   
3. Запустите приложение:
   
   python main.py
   



- Настройка длины пароля (8–32 символа).
- Выбор типов символов.
- Генерация криптографически стойких паролей.
- Сохранение истории последних 20 паролей.
- Копирование пароля в буфер обмена.
